## 🎯 O Diferencial do Projeto

Este portfólio foi desenhado para fugir do padrão genérico. A interface transmite um posicionamento sênior e minimalista, refletindo a mentalidade de quem usa a tecnologia não apenas para escrever código, mas para **crescer negócios e otimizar resultados**.

### ✨ Features de UI/UX
* **Dark Mode Profissional:** Paleta de cores baseada em tons profundos de azul e cinza, focada em legibilidade e contraste.
* **Gooey Mouse Effect:** Um fundo interativo onde bolhas reagem dinamicamente e se fundem ao movimento do cursor do usuário, construído com filtros SVG complexos e otimizado via JavaScript.
* **Botões Magnetic:** Interação premium que atrai o botão suavemente na direção do mouse, utilizando cálculo de bounding box e `requestAnimationFrame`.
* **Scroll Reveal:** Elementos surgem suavemente na tela à medida que o usuário rola a página, impulsionado pela API nativa `IntersectionObserver`.
* **Totalmente Responsivo:** Layout fluido que se adapta perfeitamente a dispositivos móveis, tablets e monitores ultrawide.

## 🛠️ Stack Tecnológica

O projeto foi construído "Vanilla" (sem frameworks pesados), priorizando a performance extrema, controle absoluto do DOM e carregamento instantâneo.

* **HTML5 Semântico**
* **CSS3** (Variáveis nativas, Flexbox, CSS Grid, Keyframes, SVG Filters)
* **JavaScript (ES6+)**
* **Ícones:** [Devicon](https://devicon.dev/) e SVG Inline
